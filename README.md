## Boilerplate Overview
An easy and simple way to configure react-native-web with webpack-dev-server and react-hot-loader.

Note: TicTacToe.js is taken simply as a working example

## Installation

1. `npm start`
2. Open `http://localhost:3000/``

## API Reference
